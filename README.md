# Kinvo - Desafio Back-end

## Rodando a aplicação

```bash
yarn dev
```

ou

```bash
npm run dev
```

## Testando a aplicação

```bash
yarn test
```

ou

```bash
npm run test
```

## Rodando o lint da aplicação

```bash
yarn lint
```

ou

```bash
npm run lint
```
